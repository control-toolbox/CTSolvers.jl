# 00 — Setup docs/ Infrastructure

**Priority**: 🔴 Critical
**Depends on**: Nothing
**Ref**: `reference/02_revised_structure.md` §2, §4

## Description

Set up the `docs/` directory structure and build configuration before writing any content.

## Checklist

- [ ] Create `docs/src/` directory tree matching the revised structure:
  ```
  docs/src/
  ├── index.md
  ├── architecture.md
  ├── guides/
  │   ├── options_system.md
  │   ├── implementing_a_strategy.md
  │   ├── implementing_a_solver.md
  │   ├── implementing_a_modeler.md
  │   ├── implementing_an_optimization_problem.md
  │   ├── orchestration_and_routing.md
  │   └── error_messages.md
  └── api/
      └── (generated by api_reference.jl)
  ```
- [ ] Create/update `docs/Project.toml` with all required dependencies (Documenter, CTSolvers, NLPModelsIpopt, etc.)
- [ ] Create/update `docs/make.jl` with the `pages` structure from `reference/02_revised_structure.md` §4
- [ ] Verify `makedocs` runs without error (even with placeholder pages)
- [ ] Add placeholder content in each `.md` file (title + "TODO" marker)

## Acceptance Criteria

- `julia --project=docs docs/make.jl` completes without error
- All pages appear in the navigation sidebar
- CI build passes with placeholder content
